💻 Sistema de Folha de Pagamento em C#
Este projeto consiste em um sistema simples de folha de pagamento desenvolvido em C#, 
capaz de calcular automaticamente os descontos de INSS e IRRF a partir do salário bruto de um funcionário.

O programa solicita ao usuário o nome do funcionário e o salário bruto, 
realiza os cálculos necessários e apresenta um resumo completo da folha de pagamento, incluindo o salário líquido.

---

## 🚀 Funcionalidades

- Entrada do nome do funcionário
- Entrada do salário bruto
- Validação de dados informados
- Cálculo automático do INSS
- Cálculo automático do IRRF
- Cálculo do salário líquido
- Exibição organizada da folha de pagamento no console
- Formatação de valores monetários no padrão brasileiro (pt-BR)

---

## 🧠 Conceitos Aplicados

Este projeto foi desenvolvido com o objetivo de praticar conceitos importantes da programação em C#, como:

- Estruturas condicionais (if / else)
- Estruturas de repetição (while)

## ▶️ Como Executar o Projeto

1. Clone o repositório
2. Acesse a pasta do projeto
3. Abra o arquivo TP01.sln no Visual Studio 2022.
4. Execute o programa pressionando F5 ou clicando em Iniciar no Visual Studio.
5. No console, informe:
   - Nome do funcionário
   - Salário bruto
6. O sistema irá calcular automaticamente:
   - Valor do INSS
   - Valor do IRRF
   - Salário líquido
